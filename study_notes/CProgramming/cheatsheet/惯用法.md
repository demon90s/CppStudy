# 惯用法

---

**判定变量是否落在某个数值范围内（p48）**

```c
if (0 <= i && i < n) { /* ... */ }

/* 相反的情况（i在范围之外） */
if (i < 0 || i >= n) { /* ... */ }
```

**for 语句的惯用法（p66）**

```c
/* 从0向上加到n-1 */
for (i = 0; i < n; ++i) { /* ... */ }

/* 从1向上加到n */
for (i = 1; i <= n; ++i) { /* ... */ }

/* 从n-1向下减到0 */
for (i = n-1; i >= 0; --i) { /* ... */ }

/* 从n向下减到1 */
for (i = n; i > 0; --i) { /* ... */ }
```

**用 for 语句建立无限循环（p67）**

```c
for (;;) { /* ... */ }
```
